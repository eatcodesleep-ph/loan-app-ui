
# Loan App Web (.NET 8 Razor Pages)
A web app for Loan Quotation and Application

## Prerequisites
- .NET 8 SDK

## Configure
Add User Secrets:
- Manage User Secrets in Visual Studio
- Copy and Paste the secret config sent on the email (Filename: LoanAppApi-Secrets.txt)

# Run the application
1. Open the solution in Visual Studio 2022 or later
2. Set the Startup Project to LoanApp.Web
3. Run the application (F5)
4. The web app will be accessible at https://localhost:{port}/
